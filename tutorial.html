Windows 98 VirtualBox Installation Guide

In this tutorial I will teach you how to properly install Windows 98 SE on a Virtual Machine, so you can use vintage software and play retro videogames with no problem. I decided to create this since there are a few tutorials on the internet but I feel like they're confusing or incomplete, since most of them don't teach you how to install the video and audio drivers. I'll promise to keep it as simple as I can!

You need these things first

  • A PC
  • Internet connection
  • Windows 98 ISO
  • Windows 98 drivers
    • Optional
    • Videogames
    • Useful links
First of all you need to install VirtualBox on your PC, mine one has Intel but I've heard AMD has some problems with virtualization.
